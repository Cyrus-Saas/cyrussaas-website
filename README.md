# CyrusSaaS Website

This is the official website for CyrusSaaS, built with Vite, React, TypeScript, and Tailwind CSS.

## Project Info

**URL**: https://cyrussaas.com (Replace with actual URL if different)

## Development

Prerequisites: Node.js & npm installed.

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
